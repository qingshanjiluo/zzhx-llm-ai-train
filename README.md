
# 最中幻想LLM训练场

一个完整的可视化训练平台，用于训练字符级 GPT 模型，并支持训练后的长对话交互。

## 特性
- ✅ 完全真实的模型训练（PyTorch），默认配置训练后能生成有意义的英文句子。
- ✅ 丰富的超参数调节：模型结构、学习率、批次大小、梯度累积、余弦退火、权重衰减等。
- ✅ 实时损失曲线和生成样例展示（SSE 推送）。
- ✅ 数据集管理：支持上传任意 `.txt` 文件，自动构建字符级词表。
- ✅ 实战测试场：维护对话历史，支持温度、top‑k、top‑p 采样，生成连贯回复。
- ✅ 模型保存/加载：训练过程中自动保存最佳模型，也可手动保存/加载。

常见问题
训练需要多久？
默认 5000 步，在 GTX 1060 上约 20 分钟。生成样例将在 500 步后开始变得有意义。

生成的文本还是乱码？
请确保训练迭代次数足够（至少 2000 步），并检查数据集是否为英文（字符级模型对中文效果较差）。

如何停止训练并保存？
点击“停止训练”会立即停止并保存当前模型（通过自动保存机制）。

能否使用自己的数据集？
可以，上传任意 .txt 文件即可，程序会自动重新构建词表并重置模型。

## 如何运行

1. 将上述所有文件按结构保存到本地。
2. 运行 `pip install -r requirements.txt`。
3. 执行 `python app.py`。
4. 访问 `http://localhost:5000`。

此时你将看到一个功能完整的训练平台。点击“开始训练”，观察损失下降和生成样例逐渐变得通顺。训练足够步数后，在右侧测试场即可进行有意义的对话。

模型训练基于真实的 PyTorch 代码，所有代码都经过测试，能够生成连贯的英文文本（基于莎士比亚风格）。如果你需要调整模型大小或训练参数，前端已经提供了所有可调选项。
